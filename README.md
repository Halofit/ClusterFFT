# ClusterFFT

Projekt pri ZZRS, ki smo ga delali v letiku 2015/16, na FRI. Repozitorij vsebuje klient, strežnik in MPI program ki ga klièe strežnik. Program uporabi MPI da razdeli glasbeno datoteko (v WAV formatu) in na njej izvede Hitro fourierjevo transformacijo (FFT). Klient in strežnik sta namenjena za izmenjavo WAV datotek.
